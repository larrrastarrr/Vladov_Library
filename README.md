# Vladov_Library

Този проект представлява проста библиотечна система, реализирана на C++, която демонстрира основни принципи на обектно-ориентираното програмиране (ООП).

## Класове

1.  **Author**: Съхранява информация за авторите (име, година на раждане). Демонстрира валидация на данни.
2.  **Book**: Представлява книга в библиотеката.
    * Използва "Rule of 5" (конструктори за копиране/преместване, деструктор).
    * Следи броя на активните инстанции чрез статична променлива `totalBooks`.
3.  **Member**: Читател на библиотеката.
4.  **Loan**: Запис за заемане, свързващ книга и читател с дати.
5.  **Library**: Управляващ клас (Manager), който съхранява списъци от книги, членове и заеми.

## Изисквания

* Компилатор поддържащ C++11 или по-нов (препоръчително C++17).

## Компилация и стартиране

Ако имате инсталиран `g++` (GCC), използвайте следната команда в терминала:

```bash
g++ -std=c++17 -Wall -Wextra -O2 -o library main.cpp LibrarySystem.cpp
